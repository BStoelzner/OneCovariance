OneCovariance code by KiDS for kids and adults
===================================

The OneCovariance code is a python package to calculate covariance matrices
for observables of photometric large-scale structure surveys.

Support
-------
If you are encountering any problem with OneCovariance, please raise an issue on the OneCovariance `GitHub <https://github.com/rreischke/OneCovariance/issues>`_ page, you can also directly send a mail to reischke@posteo.net.
